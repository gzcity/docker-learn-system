# Docker 源码学习系统 — 长期记忆

## 系统信息
- **项目**：Docker 源码学习系统（智能体原生学习系统）
- **设计文档**：`docker-learn-system/README.md`、`ARCHITECTURE.md`、`ROADMAP.md`
- **引擎核心**：`engine/knowledge_graph.py`（知识图谱）+ `engine/orchestrator.py`（编排器）+ `engine/knowledge_base.py`（知识库/书籍/笔记）
- **入口**：`learn.py`（交互式命令行）
- **部署**：`deploy/` 目录（Docker Compose 方案）

## 知识图谱
- 从 `docker-knowledge/CONCEPTS.md` 加载，当前 **16 个概念**
- 每个概念包含：定义、关联、代码引用、误解列表
- 20 条常见误解纠正（分布在 8 个概念中）
- 支持：概念检索、关联查询、学习路径 BFS、误解检测

## 学习状态
- 掌握度存储在 `memory/mastery.json`（自动持久化）
- 每日学习日志在 `memory/YYYY-MM-DD.md`
- 支持间隔重复（改进 SM-2 算法）
